# Juego Duck Hunt en C
Este repositorio alberga el código que da solución al proyecto final de la materia Programación Estructurada.
EL programa ha sido desarrollado en su totalidad en lenguaje C.
Además, para fines de entretenimiento, la forma de juego conlleva el uso de un mando externo, previamente configurado en Arduino para la correcta comunicación del serial y del programa.
